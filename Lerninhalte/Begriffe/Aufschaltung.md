# Synonyme

- **Anbindung**
    -> Wird verwendet, wenn eine Anlagen- oder Gerätetechnik technisch an ein übergeordnetes System angeschlossen wird.

- **Integration**
    → Benutzt man, wenn verschiedene Systeme oder Komponenten miteinander verbunden und aufeinander abgestimmt werden, sodass sie gemeinsam funktionieren.

- **Weiterleitung von Störmeldungen**  
    -> Verwendet man, wenn die Störmeldung von einer Quelle (z. B. einer Maschine) zur zentralen Überwachung weitergereicht oder übertragen wird.

- **Übertragung an Zentrale**  
    -> Dieser Begriff wird genutzt, wenn Informationen oder Störmeldungen gezielt an eine zentrale Stelle zur weiteren Bearbeitung oder Überwachung geschickt werden.

# Begriffserklärung

Etwas aufzuschalten heißt, etwas mit einem anderen System zu verbinden, damit dort Infos oder Signale ankommen und genutzt werden können.
