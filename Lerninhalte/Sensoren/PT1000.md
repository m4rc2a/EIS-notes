# Begriffserklärung
[[PT1000]] ist ein [[Temperatursensor]] in Form eines PTCs aus Platin.
# Parameter
-  Messstrom ($I_\text{Messung}$) ^Messstrom

> **Warnung**:
> Der [[PT10000|PT1000 Sensor]] darf nicht mit zu hohen Messstrom ($I_\text{Messung}$) durchlaufen werden.
# Konstanten

| Eigenschaft                                                      | Wert                                   |
| ---------------------------------------------------------------- | -------------------------------------- |
| Temperaturkoeffizient ($\alpha_{\mathrm{T100}}$)                 | $3{,}85 \times 10^{-3}\ /\ \mathrm{K}$ |
| Widerstand bei $0\,^\circ\mathrm{C}$ ($R_{\mathrm{T100},0}$)     | $1000\,\Omega$                         |
| Widerstand bei $100\,^\circ\mathrm{C}$ ($R_{\mathrm{T100},100}$) |                                        |
# Features
- keine Kalibrierung durch Temperaturmesskammer nötig
- wird hergestellt in zwei Varianten:
	- 2 Draht
		-> Dinge müssen beachtet werden um das Messergebnis nicht zu verfälschen
	- 4 Draht
		-> **Bei 4 Draht** Umgebung und Leitungswiderstände wirken sich nicht auf das Messergebnis aus
