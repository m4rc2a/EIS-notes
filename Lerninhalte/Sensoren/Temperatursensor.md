# Begriffserklärung
Ein [[Temperatursensor]] ist ein [[Sensor]] welcher die Temperatur erfassen kann.
# Varianten
[[Temperatursensor|Temperatursensoren]] werden anhand ihrer Funktionsweise unterschieden.
Daher gibt es unterschiedliche Arten von [[Temperatursensor|Temperatursensoren]]:
- [[Thermistor]]
- [[Widerstandstemperatursensor]]
- Thermoelement
- Halbleiter-Temperatursensor
- Infrarot-Temperatursensor
- Flüssigkristall-Thermometer
