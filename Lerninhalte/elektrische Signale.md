# Begriffserklärung
Ein [[elektrische Signale|elektrisches Signal]] ist eine Veränderung von elektrischer Spannung oder [[Strom]] über die Zeit hinweg, die **Informationen transportiert**.
-> vereinfacht: Es ist eine Art „Sprache“, mit der elektrische Geräte Informationen austauschen.
## Formen von [[elektrische Signale|elektrischen Signalen]]
Es gibt unterschiedliche Formen von elektrischen Signalen
- Digitale Signale
	- Seriell
	- Parallel
- Analoge Signale
	- Spannungssignale
	- Stromsignale
	- Widerstandssignale
	- Frequenzsignale
	- Amplitude-Modulierte Signale
